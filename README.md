*Бэкенд веб-приложения для управления автостоянкой.*
Сборка и запуск:

- mvn clean package
- docker-compose up -d

API доступен по адресу: localhost:8080

Структура БД описана в файле src/main/resources/schema.sql

Тестовые данные БД загружаются через src/main/resources/data.sql
